# Experiment History

These scripts are the project search history. They are kept for transparency because the professor asked to include what was tried, but they are not required for the final model run.

Use the root-level final pipeline for the submitted model:

```bash
python FINAL_MODEL_PIPELINE.py
```

Use the heavier optional environment only if rerunning experiments:

```bash
pip install -r requirements-experiments.txt
```

Most experiment outputs are saved in the root-level `experiment_results/` folder.
